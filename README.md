# 正则表达式可视化编辑器

一个无需后端、可直接托管在 GitHub Pages 的正则表达式可视化编辑器。支持常见语法片段一键插入、实时语法解析、匹配结果预览与替换操作，让正则调试更直观。

## 主要特性

- 🧱 **所见即所得的语法构建器**：通过工具栏快速插入或包裹常见语法，如分组、前瞻/后顾、字符集等。
- 🔍 **实时匹配预览**：对测试文本即时高亮匹配结果，并展示捕获组详情。
- 🧠 **语法可视化说明**：按 token 列出语法节点与释义，帮助理解复杂表达式。
- 🔄 **替换模拟**：提供替换模板输入框，支持 `$1`、`$&` 等占位符，便于验证替换结果。
- 🌓 **自适应深浅色模式**：根据系统偏好自动切换主题，保持良好可读性。

## 本地预览

直接打开 `index.html` 即可，无需任何构建或依赖。

```bash
python3 -m http.server 3000
# 浏览器访问 http://localhost:3000/
```

## 部署到 GitHub Pages

1. 将仓库推送到 GitHub。
2. 打开仓库的 **Settings → Pages**。
3. 选择 `main` 分支（或对应分支）的根目录作为发布源。
4. 保存后稍等片刻，即可通过 GitHub Pages 域名访问在线版本。

## 文件结构

- `index.html`：页面结构与默认内容。
- `styles.css`：响应式、深浅色自适应样式。
- `app.js`：正则解析、可视化、匹配与替换逻辑。
